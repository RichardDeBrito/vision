DESCRIÇÃO DO PROJETO:
O projeto se trata da criação de um Catálogo Interativo de Filmes e Séries com APIs Públicas. O objetivo é desenvolver uma aplicação Web responsiva que consuma dados de uma API REST pública, exibindo filmes e séries com filtros, detalhes e interação.

RESUMO DA APLICAÇÃO:
Construir um catálogo interativo filmes e séries que:
- Lista lançamentos recentes
- Permite paginação e filtragem
- Mostra detalhes de filmes/séries
- Exibe temporadas/episódios quando houver
- Trata erros e carrega dados com feedback visual

API's UTILIZADAS NO PROJETO:
- TMDB ( The Movies Database API )

A API TMDB é uma API REST pública e gratuita que fornece informações sobre filmes, séries e programas de TV.

INSTRUÇÔES PARA EXECUÇÂO:
Para a execução do projeto Vision, siga os passos:

1. Pré-requisitos

Antes de iniciar, certifique-se de ter:
- Um navegador atualizado (Chrome, Firefox, Edge).
- Conexão com a internet (necessária para as requisições à API utilizada).
- Arquivos do projeto baixados ou clonados do repositório no GitHub.

2. Como executar o projeto

- Baixe ou clone o repositório:
git clone https://github.com/RichardDeBrito/vision.git
- Acesse a pasta do projeto:
cd vision

3. Para abrir a aplicação:

- Localize o arquivo index.html.
- Clique duas vezes sobre ele para abrir no navegador ou
arraste-o para uma aba do navegador.
